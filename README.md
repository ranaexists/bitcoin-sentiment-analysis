# Bitcoin Market Sentiment vs Trader Performance Analysis

## Overview

This project analyzes the relationship between Bitcoin market sentiment and trader performance using:

* Bitcoin Fear & Greed Index Dataset
* Hyperliquid Historical Trading Dataset

The objective is to explore how market sentiment impacts trading outcomes, identify hidden patterns, and derive actionable trading insights.

## Analysis Performed

* Data Cleaning & Validation
* Dataset Integration
* Sentiment-wise Profitability Analysis
* Win Rate Analysis
* Profit Distribution Analysis
* Trading Activity Analysis
* Risk Behaviour Analysis
* Top Trader Analysis
* Coin-wise Performance Analysis
* Trade Size vs Profitability Analysis
* Statistical Significance Testing

## Key Findings

* Extreme Greed produced the highest average profitability.
* Fear periods showed the highest trading activity and volume.
* Top-performing traders achieved their best results during Extreme Greed conditions.
* Certain coin-sentiment combinations consistently outperformed others.
* Larger trade sizes generated significantly higher average profits.
* Statistical testing confirmed significant profitability differences across sentiment regimes.
